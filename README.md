Qasa Ambientação Digital — Projeto de Desenvolvimento Web


Este projeto marcou minha primeira experiência no desenvolvimento de soluções digitais para o mercado corporativo. O cliente, titular da Qasa Ambientação Digital, empresa especializada em design 3D de interiores voltado ao segmento de granito, apresentou a demanda de um site institucional com o objetivo de exibir seu portfólio e atrair novos clientes.

Desde o início, orientei o cliente a desenvolver uma identidade visual própria para o projeto, garantindo que o resultado final refletisse a essência e o posicionamento da marca. A proposta foi prontamente atendida, resultando em uma base visual alinhada ao perfil da empresa.

O desenvolvimento foi conduzido com as tecnologias HTML, CSS e JavaScript, aplicadas de forma estratégica em cada etapa:

HTML: Estruturado de maneira clara e semântica, com código limpo e de fácil compreensão, visando acessibilidade e manutenibilidade. 

CSS: Recebeu atenção especial, dado o segmento de atuação do cliente. O estilo foi desenvolvido com foco em elegância, precisão visual e facilidade de manutenção futura, representando fielmente o nível de qualidade dos projetos da empresa.

JavaScript: Responsável pelo principal diferencial técnico do projeto. Atendendo a uma solicitação específica do cliente, foi implementado um sistema de troca automática de imagens na seção de apresentação de serviços e planos, com funções otimizadas e uma seleção criteriosa de imagens. O resultado superou as expectativas, agregando dinamismo e sofisticação à experiência do usuário.
O bacana é que todos os botões são interativos, e tem uma API que ao clicar no botão te leva para o WhatsApp da empresa.
